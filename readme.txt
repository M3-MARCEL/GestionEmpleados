Instalación de dependencias necesarias:

pip install -r requerimientos.txt

Este comando instalará:
- mysql-connector-python: Conexión a MySQL desde Python.
- reportlab: Generación de informes en PDF.
- pandas: Exportación de informes en Excel.

Iniciar la interfaz gráfica:

Ejecutar: python interfaz.py

Herencia y Polimorfismo: La clase Departamento hereda de Empleado, demostrando herencia.
Se utiliza polimorfismo mediante el método mostrar_info() en ambas clases.
Validación de Entrada: La interfaz usa Combobox para evitar errores de entrada y facilitar la selección de datos.
